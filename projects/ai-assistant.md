# AI Assistant 🤖

## 🇬🇧 EN

### 🔥 Description
This project is about creating a simple AI assistant to help manage ideas, tasks, and projects.

### 🧠 Features (planned)
- idea organization  
- simple task tracking  
- project notes  
- AI-generated suggestions  

### 🚀 Goal
Build a simple, useful assistant for everyday use.
