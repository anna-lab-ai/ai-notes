# AI Daily Planner 📅

## 🇬🇧 EN

### 🔥 Description
This tool helps plan your day using AI.

### 🧠 How to use
1. Write your tasks  
2. Use the prompt  
3. Follow AI plan  

### 🤖 Prompt
Act as a productivity assistant.
Organize my day, prioritize tasks, and create a simple plan.

My tasks:
- [task 1]
- [task 2]
- [task 3]

### 🚀 Example

Tasks:
- clean desk  
- learn GitHub  
- test AI ideas  

AI Plan:
1. Clean desk (10 min)  
2. Learn GitHub basics (30 min)  
3. Test AI ideas (20 min)
